# Student Management System (C++)

## 📌 Description
This is a console-based Student Management System developed in C++ using file handling. It allows users to manage student records efficiently.

## 🚀 Features
- Add student records
- Display all students
- Update student details
- Delete student records
- Persistent storage using file handling

## 🛠️ Technologies Used
- C++
- File Handling (Binary Files)

## 📂 File Used
- students.dat → stores student records

## 🎯 Outcome
A functional menu-driven application for managing student data with persistent storage.
